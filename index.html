Find Stuff (WIP)

Back when I was a residence advisor for UBC, a concern I kept hearing from first year students was how hard it was to get from class to class since they had to look up each building individually and then route to it. The UBC campus is massive, so I totally understand how easy it can be to get lost.

My thought with this project was to make it as easy as possible for incoming freshmen to find stuff on campus. All they have to do is type in the course codes of all their classes, and the app automatically routes them between every course in order of their occurence. If there is a gap between classes, Find Stuff also recommends study spots that are near your last class!

I built this app with JS, React, and Node, using my custom UBC API, the Positionstack Geocoding API, and Mapbox GL. I also used material UI for a bunch of the visual things. This wildly zoomed out photo doesn't quite do the app justice, so please check out the code here and run it yourself!

Moonland

This was just a fun little summer project. I like 8-bit games so I wanted to try to make a simple one. In this game, you spawn at a certain position and have to navigate to the final destination planet while avoiding the gravitational pull of nearby planets, all while conserving fuel! The thought behind this was that you'd have to utilize the orbital mechanics to get to the end, and strategically use fuel, just like they did for the real moon landing.

This was made entirely in Python using Pygame and various other Python modules.

MINT Home

MINT is a neurotechnology focused UBC design team, and I was the lead software developer for the MINT Home project. Individuals with certain motor disabilities may have extreme difficulty performing tasks such as picking up a remote, or changing the channel of a TV. We created a wearable headset that measures brain activity to complete these actions hands-free!

On the software team, we designed the app that takes in the raw data, processes it, sends it through an ML model, takes the model output, and sends a signal to a devices connected by bluetooth. The user would wear the headset, and think of a certain stimulus, which would then trigger an action, such as turning on a TV, or changing the volume.

We made this app with Electron, JS, React, and D3 for the frontend, and a Flask/Python backend. I wish I could link the repo, but unfortunately all the code is private since it's being submitted to competitions.
