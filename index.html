Find Stuff (WIP)

This was the continuation of my prior two projects (the UBC API and course info webscraper shown below). I have a terrible sense of direction, and UBC is a huge campus, and every time I wanted to get to my class I'd have to navigate through the UBC site to find the building name, then put that into Maps, and repeat for every course..

My thought with this was it could be a easy way for directionally challenged freshmen (like I used to be..) to navigate between classes. Find Stuff also recommends study spots near your course buildings if there are gaps in your timetable! I built this with JS, React, and Node, using my UBC API, Mapbox GL for the map, and the Positionstack Geocoding API. I used material UI for a bunch of the visual things too.

This wildly zoomed out photo doesn't quite do the app justice, so please check out the code here and run it yourself!

Moonland

This was just a fun little summer project. I like 8-bit games so I wanted to try to make a simple one. In this game, you spawn at a certain position and have to navigate to the final destination planet while avoiding the gravitational pull of nearby planets, all while conserving fuel! MINT Home

MINT is a neurotechnology focused UBC design team, and I was the lead software developer for the MINT Home project. Individuals with certain motor disabilities may have extreme difficulty performing tasks such as picking up a remote, or changing the channel of a TV. We created a wearable headset that measures brain activity to complete these actions hands-free!

On the software team, we designed the app that takes in the raw data, processes it, sends it through an ML model, takes the model output, and sends a signal to a devices connected by bluetooth. The user would wear the headset, and think of a certain stimulus, which would then trigger an action, such as turning on a TV, or changing the volume.

We made this app with Electron, JS, React, and D3 for the frontend, and a Flask/Python backend. I wish I could link the repo, but unfortunately all the code is private since it's being submitted to competitions.
